About

I've always had an interest in computer technologies but never indulged myself in it seriously until the outburst of the Covid-19 pandemic. I researched different fields of computer programming and got really interested in front-end web development. What hooked me to it was that one can immediately see the results of his work, represented in a visual way and also the fact that there is design involved in it. I started messing around with HTML and CSS and discovered the great possibilities for creating dynamic web content. Naturally I started looking for more technologies required to become a front-end developer, which led me to the JavaScript technological stack and to the SoftUni JavaScript Program which I enrolled. Since September 2020 I have been learning the ins and outs of front-end development and discovered a vast universe of knowledge to explore. The more things I learned, the more new things came up to be learned and it has been a never ending process. What I lack is the experience in a real production environment.
